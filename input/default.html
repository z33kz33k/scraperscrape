How do I compare two data sets?

First, select the data range for your base data. For example, you can select all the buildings completed in the United States between 1960 and 2020.

Then select the data to compare against the base data. Continuing our example, you can select all the United Arab Emirates buildings completed between 1960 and 2020.

We've set these parameters for you to try, but feel free to explore the data by changing them.

After checking the box next to the analysis output you'd like to see, click Show Results.
